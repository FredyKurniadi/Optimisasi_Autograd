# Quick Start

1. Siapkan environment:
```powershell
./scripts/setup_all.ps1
```

2. Jalankan optimisasi multi-objective:
```powershell
./scripts/run_optimize.ps1
```

3. Lihat ringkasan metrik run terbaru:
```powershell
./scripts/show_latest_metrics.ps1
```

4. Jalankan test:
```powershell
./scripts/run_all_tests.ps1
```
